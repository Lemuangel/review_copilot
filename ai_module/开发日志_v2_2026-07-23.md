## 二、新增功能详细说明

### 2.1 RAG 向量库扩充（1015 块，576 条差评）
- 全量加载 Amazon All_Beauty 5-core 数据集（5269 条评论）
- 筛选 576 条差评（≤3 星），文本分割后 942 块 → 合并后 1015 条
- 相似度阈值优化：0.65 → 0.35（适配英文语义检索）
- 构建耗时：~270s（CPU），持久化至 `vector_db/`

### 2.2 推理延迟优化
- **流式输出**：`init_llm(streaming=True)` 降低首字延迟
- **LRU 缓存**：200 条缓存，TTL 1 小时，重复分析秒级返回
- **批量并发**：`/ai/batch` 端点，ThreadPoolExecutor 并发分析
- 缓存管理：`/ai/cache/stats` + `/ai/cache/clear`

### 2.3 翻译质量自动化评估
- **BLEU 评分**：n-gram 重叠 + 简短惩罚
- **文本完整性**：长度比、截断检测、乱码检测
- **关键信息保留率**：自动提取关键词，检查保留比例
- **语义相似度**：基于 BGE-M3 余弦相似度
- **综合评分**：0-100 分 + A/B/C/D/F 等级
- API：`/ai/eval/translation` + `/ai/eval/translation/batch`

### 2.4 Prompt 边界情况处理
- 空输入 / 纯符号 / 纯表情 / 超长 / 重复字符 / 控制字符 / 过短
- 智能截断：在句子边界处截断，保留 70%+ 内容
- 输入清洗：评分/国家/类目/ASIN 格式标准化
- 分析结果附带 `input_warnings` 字段

### 2.5 Prompt 版本管理 + A/B 测试
- **PromptVersionManager**：版本注册/切换/回滚
- **ABTestRunner**：一致性哈希分流/指标收集/综合评分/赢家推荐
- 内置 v1_original + v2_enhanced 两个版本
- API：`/ai/prompt/versions` + `/ai/prompt/activate` + `/ai/ab/report` + `/ai/ab/feedback`
